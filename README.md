# Midnight Sonata

<img width="1707" height="1067" alt="MidnightSonata" src="https://github.com/user-attachments/assets/177318e1-47ef-4cf3-895d-f57303a22c0a" />


Midnight Sonata is a monochrome icon theme to match "Darkly" theme nicely.

You can get "Darkly" theme for KDE here: https://github.com/Bali10050/Darkly

Requirements
--
Midnight Sonata icon theme require Papirus icon theme to look right: https://store.kde.org/p/1166289
